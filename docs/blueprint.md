# **App Name**: Albino Logistics

## Core Features:

- Homepage Banner: Prominent banner showcasing cargo handling services with 'Contract Services' and 'Join the Team' CTAs.
- About Albino Section: Concise institutional text focusing on Albino's experience, credibility, and values.
- Services Page: Detailed service listing: cargo handling outsourcing, trained teams, flexibility, and custom support.
- Contact Form: Simple form: name, phone, email, and message fields.
- Footer Information: Contact info, address, social media links, and company logo.
- SEO Optimization: Strategic use of keywords: 'logistics outsourcing,' 'cargo handling labor,' and 'outsourced logistics services.'

## Style Guidelines:

- Primary color: Dark Blue (#2E4A62) to convey trust and professionalism.
- Background color: Light gray (#E9ECEF), nearly white, to create a clean layout.
- Accent color: Yellow-orange (#F0AD4E) to draw attention to main calls to action.
- Body text font: 'Inter', a sans-serif, for a modern and neutral feel.
- Headline font: 'Space Grotesk', a sans-serif, for titles and CTAs. Note: currently only Google Fonts are supported.
- Vector icons representing transportation, forklifts, boxes, warehouses, and operational teams.
- Clean and modern design with ample whitespace for readability.
- Subtle loading animations on content.
